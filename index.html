Could not import cv2 in python3.5

Error infomation

ImportError: /opt/ros/kinetoc/lib/python2.7/dist-packages/cv2.so: undefined symbol: PyCObject_Type

Problem discription

The package cv2 could not be imported into python3.5, but it can be imported into python 2.7

Solution

The error appers while the python trys to import cv2.so from a wrong python path.
1- Check the current path for python
In a python console:
>>>import sys
>>>sys.path
2- Change the path
Change in the current terminal
export PYTHONPATH=/home/.../lib/python3.5/site-packages
